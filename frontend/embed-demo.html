Cero Embed Demo

This page demonstrates the two-line installation snippet. On a production site, replace the script source with the hosted widget URL and set the asset base URL explicitly: 

<script src="https://chatbot.plataformacero.org/cero-widget.iife.js"></script>
<cero-chat-widget api-base-url="https://chatbot.plataformacero.org" asset-base-url="https://chatbot.plataformacero.org"></cero-chat-widget>

Keep the script synchronous when relying on automatic asset origin detection. If you must load it with async or defer, always provide asset-base-url so the widget knows where to fetch icons.
