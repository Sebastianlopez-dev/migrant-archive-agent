Problem

Long videos bury the knowledge.

Specific context is hard to find, translate,
and reuse when important migrant narratives are locked
inside long talks and weak captions.

  • Specific answers
  • Non-mainstream narratives
  • Spanish + Catalan access
  • Weak YouTube captions

Solution

Find specific context in long videos.

Cero helps people find specific moments, surface underrepresented narratives, and get source-backed quotes with exact timestamps.

  • Catalog tools: Find the right video
  • Semantic search: Retrieve the exact moment
  • Memory: Continue follow-up questions
  • Multilingual access: 5 languages supported
  • Voice input: Accessibility

Agent structure

def create_agent():
    llm = ChatGoogleGenerativeAI(
        model="gemini-2.5-flash",
        temperature=0.2,
        tool_config={"function_calling_config": {"mode": "ANY"}},
    )

    store = Chroma(
        collection_name="migrant_archive",
        embedding_function=GoogleGenerativeAIEmbeddings(
            model="gemini-embedding-2"
        ),
        persist_directory="data/chroma",
    )

    tools = [
        list_videos,
        get_video_info,
        search_transcripts,
    ]

    prompt = system_prompt + chat_history + user_input + agent_scratchpad

    agent = create_tool_calling_agent(llm, tools, prompt)

    executor = AgentExecutor(
        agent=agent,
        tools=tools,
        max_iterations=10,
        return_intermediate_steps=True,
    )

    return RunnableWithMessageHistory(
        executor,
        get_session_history,
        history_messages_key="chat_history",
    )

Product surface

FastAPI service, typed frontend.

  • Ask — grounded answer with sources POST /api/ask
  • Speak — audio to text, Groq Whisper POST /api/transcribe
  • Reset — clear conversation memory DELETE /api/session/{id}

The browser never runs the AI. It calls the API; a FastAPI service runs the agent on the backend and returns structured JSON.
