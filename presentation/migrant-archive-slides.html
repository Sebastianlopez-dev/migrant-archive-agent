Migrant Archive

Sebastian Lopez Largo

Problem

Long videos bury the knowledge.

Specific context is hard to find, translate, and reuse when important migrant narratives are locked inside long talks and weak captions.

specific answers non-mainstream narratives Spanish + Catalan access weak YouTube captions

Solution

Find specific context in long videos.

Cero helps people find specific moments, surface underrepresented narratives, and get source-backed quotes with exact timestamps.

Catalog tools — find the right video Semantic search — retrieve the exact moment Memory — continue follow-up questions Multilingual access

Colab GPU faster-whisper local YouTube captions VideoData JSON

The rest of the system does not care how a transcript was produced.

Decision

Native tool calling beat ReAct parsing.

ReAct text parsing ~30% failure rate native tool calls structured actions

This is the engineering decision that made the Spanish agent reliable enough to demo.

Quality

High faithfulness. Recall is next.

Metric Score Meaning
Faithfulness 0.96 Claims backed by context
Answer relevancy 0.78 Answers the question
Context precision 0.90 Retrieved context is relevant
Context recall 0.73 Ground truth is covered

High faithfulness means Cero rarely invents. Context recall is the next improvement target.
