S02 · Chunking

1 000 token chunks · 200 token overlap.

Captures ~4–5 min of speech — one complete idea. 20% overlap bridges ideas across chunk boundaries.

title + description + [MM:SS] enriched text per chunk ~12 chunks / 1-hour video

S02 · Embedding providers

Dependency inversion. Two providers.

gemini-embedding-001 — 3072d BGE-M3 local — 1024d

One abstract contract. Two implementations. Zero coupling.

S03 · Vector store

ChromaDB — zero-config, no API key.

Persistent local storage. Semantic search. Metadata filters by video, speaker, and timestamp.

ChromaDB zero-config local

S04–S05 · Verify then query

First verify the data. Then query with memory.

S04 extract_sample.py — verify chunks arrived intact

S04 validates integrity. S05 shows the progression: keyword → semantic → memory.
